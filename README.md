# SimpleRedis

#### 描述
使用electron-vue和node.js开发的一款redis数据管理客户端，支持多平台

#### 安装

1.  安装依赖包
    - npm install(建议使用yarn install)
2.  build生成安装程序
    - npm run build
    - 注：编译好的安装包在 build/apps 目录下
3.  安装运行
